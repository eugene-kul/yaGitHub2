ОСОБЕННОСТИ ФУНДАМЕНТА НА СВАЯХ ДЛЯ ПИРСА

Построить хороший пирс достаточно сложно. Основание для пирса должно иметь максимальную защиту от коррозии и выдерживать серьёзные нагрузки.

Для возведения пирса идеально подойдут винтовые сваи с полимерным антикоррозийным покрытием.

Важно! Винтовые сваи для пирса устанавливаются только с привлечением спецтехники. Вручную установить такое основание невозможно.

Обычно для строительства пирса используют опоры с диаметром 108 мм. Они выдерживают большие нагрузки, оставаясь при этом устойчивыми.

ЭТАПЫ ВОЗВЕДЕНИЯ ФУНДАМЕНТА ДЛЯ ЗАБОРА:

  • Осмотр территории (есть ли перепады высот);
  • Анализ подводного грунта;

  • Монтаж свай;
